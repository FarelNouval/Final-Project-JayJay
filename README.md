# Automation Test Framework (Web UI + API)

## Tech Stack
- Java 17
- Gradle
- Selenium
- Cucumber (Gherkin)
- JUnit
- WebDriverManager
- GitHub Actions (CI/CD)

## Test Type
- Web UI
- API (Coming Soon)

## Target Website
https://www.saucedemo.com/

## How to Run Web Test / Api Test
./gradlew webTest
./gradlew apiTest

## Test Scenarios
- Login (valid & invalid)
- Add product to cart
- Filter product (Price & Name)
- Logout

## Reports
HTML : reports/web-cucumber-report.html
JSON : reports/web-cucumber-report.json
